## Production Notes

### Visual Style

| Element | Specification |
|---------|---------------|
| **Color Palette** | Depot brand colors (purple/blue gradient), clean white backgrounds |
| **Typography** | Modern sans-serif (Inter, SF Pro, or similar) |
| **Graphics** | Clean diagrams, subtle animations, not overly technical |
| **Screen Captures** | 1080p minimum, smooth mouse movement, highlighted key areas |

### Audio

| Element | Specification |
|---------|---------------|
| **Voiceover** | Professional, clear, slightly energetic but not over-the-top |
| **Background Music** | Subtle, upbeat tech-style, fades during comparisons |
| **Sound Effects** | Minimal — clicks, success chimes, nothing distracting |

### On-Screen Text

Keep text minimal and readable:
- Title: 48-60pt
- Body: 24-32pt
- Contrast ratio: 4.5:1 minimum

### Motion Graphics

| Element | Timing | Style |
|--------|--------|-------|
| Transitions | 0.3-0.5s | Smooth fades, slides |
| Builds | 1-2s | Progressive reveals |
| Comparisons | Real-time | No speed-up during live comparison |
